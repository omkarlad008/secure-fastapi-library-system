# Secure FastAPI Library System

A security-focused web application built with **FastAPI** that demonstrates user authentication, OTP verification, password reset, role-based access control, and protected file download concepts in a library management scenario.

This project was developed as part of an academic assignment focused on secure system design, backend development, and practical security implementation.

---

## Project Overview

The application allows users to:

- register with a valid email and strong password
- verify their account using OTP
- log in securely
- reset their password using a token-based flow
- browse and search for books
- download content with file integrity verification concepts
- access role-based features for admin and standard users

The project was designed to combine usability with backend security concepts in a simple, functional web application.

---

## Key Features

- User registration with password validation
- OTP-based account verification
- Secure login flow
- Forgot password / reset password functionality
- Role-based access control for admin and user views
- File integrity demonstration using MD5 checksum
- Secure communication concepts using HTTPS / SSL
- Token-based authentication and protected access patterns
